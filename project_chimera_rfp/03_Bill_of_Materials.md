# Project Chimera: Bill of Materials (BOM)

This document outlines the critical components and materials required for the fabrication of a single Project Chimera prototype.

| Item ID | Component/Material | Specification | Quantity | Notes |
|---|---|---|---|---|
| **1.0** | **Airframe** | | | |
| 1.1 | Carbon Fiber Reinforced Polymer (CFRP) | Aerospace Grade, High Modulus | 8,000 kg | For primary fuselage and skin. |
| 1.2 | Titanium Alloy | Ti-6Al-4V | 1,500 kg | For structural hardpoints and engine mounts. |
| **2.0** | **MHD Propulsion System** | | | |
| 2.1 | Superconducting Magnets | YBCO (Yttrium Barium Copper Oxide) Tape | 8 units | Toroidal configuration, rated for >5.0 Tesla. |
| 2.2 | Cryocooler System | Gifford-McMahon Cryocoolers | 8 units | To maintain superconducting temperature (40-77K). |
| 2.3 | RF Ionization Emitters | 2.45 GHz Magnetron-based | 16 units | Redundant pair for each of the 8 segments. |
| 2.4 | Electrode Material | Hafnium-Carbide Composite | 32 units | High-erosion resistance for plasma channels. |
| **3.0** | **Power System** | | | |
| 3.1 | Compact Nuclear Reactor Core | Lead-Cooled Fast Reactor (LFR), 50 MWth | 1 unit | Includes fuel, control rods, and primary vessel. |
| 3.2 | Brayton Cycle Turbine | Closed-loop Helium Turbine | 1 unit | For converting thermal to electrical power (20 MWe). |
| 3.3 | Radiation Shielding | Borated Polyethylene & Depleted Uranium | 2,500 kg | For reactor containment. |
| 3.4 | Superconducting Busbars | Magnesium Diboride (MgB2) | 200 meters | For high-voltage power distribution. |
| **4.0** | **Avionics & Control** | | | |
| 4.1 | Flight Control Computer | Triple-Redundant, DO-178C Level A certified | 1 set | |
| 4.2 | Inertial Navigation System (INS) | Ring Laser Gyro, GPS-aided | 3 units | Redundant set. |
| 4.3 | Sensor Suite | High-temperature pressure and magnetic field sensors | 1 set | For real-time feedback from MHD channels. |
